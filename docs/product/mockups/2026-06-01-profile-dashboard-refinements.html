Profile Dashboard refinements

Three directions starting from the current Circle Dashboard. The shared move is to make profile management feel calm and modern: lighter active states, one focused edit area, compact add affordances, and sign out treated as account metadata instead of a big competing action.

Option A: Soft dashboard, inline edit

Best near-term implementation. Existing profiles feel like selectable records, the selected one opens a single edit panel, and adding is a compact row until tapped. This removes the black slab while keeping the current information architecture.
