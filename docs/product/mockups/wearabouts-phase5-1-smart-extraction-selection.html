Smart extraction should prepare only the garment the user actually wants.

The right default is not "generate the outfit." It is "scan the photo, identify what is new or requested, and prepare only those pieces." This prevents the same jeans or shoes from being regenerated across ten different T-shirt photos.

1. Upload starts with intent, not source type

The user chooses the job: pick after scan, new tops, new bottoms, or core outfit. Shoes are off by default.
