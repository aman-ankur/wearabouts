User stories

What the MVP must let people do.

Each story maps to one or more screens below. The visual language is intentionally clean, quiet, and modern: mostly black, white, grey, and garment color as the accent.

As a household owner

I can create profiles for myself and my wife so wardrobes and trip looks stay separate but coordinated.

As a new user

I can use demo mode to test the whole product flow without spending AI credits.

As a wardrobe builder

I can upload outfit or item photos and receive standardized closet items for review.

As a quality-conscious user

I can retry, delete, rename, and edit detected items before they enter the closet.

As a style explorer

I can swipe through tops, bottoms, shoes, and accessories on my body photo to build looks quickly.

As a traveler

I can create a trip and get day-by-day outfits that consider weather, activities, style, and baggage mode.

As a practical packer

I can approve trip looks and get a packing list derived from the clothes actually used.

As a visual person

I can generate a polished avatar render for a selected outfit instead of paying to render every swipe.

As a repeat user

I can mark outfits as worn, skipped, or changed so future suggestions become more personal.
